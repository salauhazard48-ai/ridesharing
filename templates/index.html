How It Works
Two ways to get moving
01
🚗

Driver Posts a Trip

Drivers select their route and seats. The system instantly calculates the optimal path and fair price per seat using Dijkstra's algorithm.

02
🔍

Passenger Searches

Passengers search by route, time, seats and budget. The five-stage matching algorithm ranks every available trip by fit score.

03
📋

Passenger Requests a Ride

No driver available? Post a ride request. Drivers see all open requests and can accept — the system auto-creates the booking instantly.

04
✅

Trip Confirmed

Once matched or accepted, both parties see the confirmed booking with route, fare, and departure details on their dashboard.

Technology
Built on intelligent algorithms
🗺️

Dijkstra's Shortest Path

Every route is computed using Dijkstra's graph algorithm across a weighted network of Nigerian cities — guaranteeing the most optimal path every time.

🎯

Five-Stage Matching

Trips are filtered by route, time window, seat availability, and budget — then ranked by a composite fit score weighted across all four dimensions.

💰

Proportional Fare Sharing

Fares are calculated dynamically based on route distance and divided proportionally among passengers — fair pricing for everyone on every trip.
